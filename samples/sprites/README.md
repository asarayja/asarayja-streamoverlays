# Sprite samples

Test sheets for the **Import sprite / GIF** feature in the editor. Both import
with the frame grid and background handled automatically.

| File | Grid | Background |
|------|------|------------|
| `walk-green-10frames.webp` | 10×1, 10 frames | Green screen — auto-removed (chroma key + despill) |
| `walk-transparent-16frames.png` | 4×4, 16 frames | Already transparent — left as-is |

## How to use

1. Editor → **Add** panel → **Import sprite / GIF** → pick a file here.
2. The grid (Cols / Rows / Frames) and background removal are detected on import.
3. Pick a **Movement** preset (e.g. *Walk left / right*) and place it.
